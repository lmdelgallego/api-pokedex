## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Requirement

- Install NestJs globally

    ```bash
      npm i -g @nestjs/cli
    ```

- Up data base
  - Install [docker desktop](https://www.docker.com/products/docker-desktop/)
  - Run docker compose

    ```bash
      docker compose up -d
    ```

## Installation

```bash
pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
